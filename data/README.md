# Data

This directory contains all data, used as imput and generated by the analyses presented in the Hobbs _et al._ manuscript.

After running the analysis, this directory will contain several additional dirs:

```bash
.
└── data
    ├── genomic_accessions
    │   ├── pectobact_accessions
    │   └── pecto_dic_accessions
    ├── pectobact
    │   ├── genomes
    │   │   └── *.fna
    │   ├── proteomes
    │   │   └── *.faa
    │   └── cazomes
    │       ├── pecto_fam_genomes
    │       └── pecto_fam_genomes_proteins
    ├── pecto_dic
    │   ├── genomes
    │   │   └── *.fna
    │   ├── proteomes
    │   │   └── *.faa
    │   ├── cazomes
    │   │   ├── pd_fam_genomes
    │   │   └── pd_fam_genomes_proteins
    │   └── tree
    │       ├── genomes
    │       │   ├── proteins
    │       │   │   └── *.faa
    │       │   ├── cds
    │       │   │   └── *.fna
    │       │   └── gbk
    │       │       └── *.gbf
    │       ├── orthologues
    │       ├── sco_proteins_aligned
    │       ├── sco_cds
    │       ├── sco_cds_aligned
    │       ├── concatenated_cds
    │       │   ├── concatenated.fasta
    │       │   └── concatenated.part
    │       └── tree
    │           ├── 01_check
    │           ├── 02_parse
    │           ├── 03_infer
    │           └── 04_bootstrap
    └── README.md
```
